# 被动收入追踪器

> 多渠道被动收入追踪与管理工具，可视化分析与趋势预测

---

## Features / 功能特点

| 功能 | 说明 |
|------|------|
| 多来源录入 | 支持课程、电子书、广告、咨询、其他等收入来源 |
| 趋势图表 | Chart.js折线图展示月度收入变化趋势 |
| 预测曲线 | 基于线性回归算法的下月收入预测 |
| 增长率计算 | 自动计算月环比增长率 |
| 来源分布 | 饼图可视化各收入来源占比 |
| 各来源趋势 | 对比各渠道收入变化的独立折线图 |
| 收入分析 | 各来源增长率一览表 |
| 数据筛选 | 按月份/年份/来源灵活查看历史数据 |
| 本地存储 | localStorage 数据持久化，无需服务器 |

## Installation / 安装

无需安装，直接在浏览器中打开 `index.html` 即可使用。

```bash
# 克隆仓库
git clone https://github.com/yourusername/passive-income-tracker.git

# 进入目录
cd passive-income-tracker

# 在浏览器中打开
open index.html  # macOS
# 或双击 index.html 文件
```

## Usage / 使用方法

### 基础用法

1. 打开 `index.html` 文件
2. 在「新增」Tab 录入某月的被动收入
3. 查看「仪表盘」获得全面的收入分析和预测
4. 使用「收入记录」筛选查看历史数据

### 录入收入示例

```javascript
// 收入记录数据结构
{
  id: 1704067200000,
  source: "课程",       // 来源类型
  amount: 5000,         // 收入金额
  month: 1,             // 月份
  year: 2024,           // 年份
  note: "线上课程销售",  // 备注（可选）
  createdAt: "2024-01-01T00:00:00.000Z"
}
```

### 支持的收入来源

| 来源 | 说明 | 推荐场景 |
|------|------|----------|
| 课程 | 知识付费课程销售 | 线上课程、训练营 |
| 电子书 | 电子出版物收入 | 付费文档、电子书 |
| 广告 | 广告分成收入 | 公众号、网站广告 |
| 咨询 | 咨询服务收入 | 付费咨询、顾问服务 |
| 其他 | 其他被动收入 | 版税、联盟分成等 |
