# 被动收入追踪器

副业收入不稳定？
**追踪+预测帮你规划被动收入。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## 收入不再"忽高忽低"

- **里程碑庆祝**：confetti动画效果，激励持续
- **效率分析**：时薪+被动指数，量化收入质量
- **趋势预测**：线性回归预测未来收入
- **多源管理**：管理多个收入来源

## 怎么用

打开 `index.html` → 添加收入来源 → 查看趋势预测
